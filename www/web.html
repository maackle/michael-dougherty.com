Recent Work

Grove Wood Print Case Customizer

A collaboration with local badass Yuji Tomita, built for local iPhone case artisans Grove. Users can import their instagram photos or upload images directly to be printed on a blank wooden iPhone case, adjusting the position and size with an easy drag-and-drop interface. The app sends the final result directly to Grove for printing.

Try it out! (We're pretty proud of this one.)

Making Room for Manufacturing

This site is the public-facing facet of an ongoing research project at University of North Carolina which seeks to explore the rapid deindustrialization of cities. It features a custom-built layered map application which visualizes the data being researched alongside other related data.

currently in development

Nikhil Kaza

Extremely customized personal website of UNC professor Nikhil Kaza. It features an extensive CMS, a project wiki, tumblr integration, and automated bibliography generation. Fully loaded!
